72tn The use of a numerical ladder as we have here – “six // seven” is frequent in wisdom literature to show completeness. See Prov 6:16; Amos 1:3, Mic 5:5. A number that seems to be sufficient for the point is increased by one, as if to say there is always one more. By using this Eliphaz simply means “in all troubles” (see H. H. Rowley, Job [NCBC], 56).
