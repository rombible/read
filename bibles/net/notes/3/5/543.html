43tn Heb “trespasses a trespass” (verb and direct object from the same Hebrew root, מַעַל, ma’al); cf. NIV “commits a violation.” The word refers to some kind of overstepping of the boundary between that which is common (i.e., available for common use by common people) and that which is holy (i.e., to be used only for holy purposes because it has been consecrated to the Lord, see further below). See the note on Lev 10:10.
