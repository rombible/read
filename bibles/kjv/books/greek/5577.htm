Strong's Concordance
pseudomarturia: false testimony
Original Word: ψευδομαρτυρία, ας, ἡ
Part of Speech: Noun, Feminine
Transliteration: pseudomarturia
Phonetic Spelling: (psyoo-dom-ar-too-ree'-ah)
Definition: false witness, false testimony
Usage: false testimony, false witness.
Thayer's Greek Lexicon
STRONGS NT 5577: ψευδομαρτυρία

ψευδομαρτυρία, ψευδομαρτυριας, ἡ, (ψευδομαρτυρέω), false testimony, false witness: Matthew 15:19; Matthew 26:59. (Plato, Plutarch; often in the Attic orators.)
