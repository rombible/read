Owl. [N] [E]

A number of species of the owl are mentioned in the Bible, ( Leviticus 11:17 ; 14:16 ; Isaiah 14:23 ; 34:15 ; Zephaniah 2:14 ) and in several other places the same Hebrew word is used where it is translated ostrich. ( Job 30:29 ; Jeremiah 50:39 ) Some of these species were common in Palestine, and, as is well known, were often found inhabiting ruins. ( Isaiah 34:11 Isaiah 34:13-15 )
